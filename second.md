# AMA - 03/03/2026

**Q1. Why is DOCTYPE used in HTML?**  
Ans: DOCTYPE is used to tell the browser which version of HTML is being used. It helps the browser render the page correctly and follow proper standards.

---

**Q2. What are Semantic Tags in HTML?**  
Ans: Semantic tags clearly describe the meaning of the content. Examples are `<header>`, `<footer>`, `<section>`, `<article>`, `<nav>`. They make code more readable and improve SEO.

---

**Q3. Difference between DROP and TRUNCATE?**  
Ans: DROP removes the entire table including its structure. TRUNCATE removes all records from the table but keeps the table structure.

---

**Q4. What are ol and ul in HTML?**  
Ans: `<ol>` is an ordered list (numbers). `<ul>` is an unordered list (bullets).

---

**Q5. Difference between Margin and Padding?**  
Ans: Margin is the space outside an element. Padding is the space inside the element between the content and border.

---

**Q6. How to create links in HTML?**  
Ans: Links are created using the `<a>` tag.  
Example:  
`<a href="https://example.com">Visit</a>`

---

**Q7. If both inline and internal CSS are applied, which works?**  
Ans: Inline CSS has higher priority than internal CSS, so inline CSS will work.

---

**Q8. How to change datatype in SQL?**  
Ans: We use ALTER TABLE with MODIFY or ALTER COLUMN depending on the database.  
Example:  
`ALTER TABLE table_name MODIFY column_name datatype;`

---

**Q9. What is the difference between Tag and Element?**  
Ans: A tag is the keyword inside angle brackets like `<p>`. An element includes the opening tag, content, and closing tag like `<p>Hello</p>`.

---

**Q10. What is Transition in HTML/CSS?**  
Ans: Transition is used in CSS to create smooth changes between property values, like color or size, over a duration.

---

**Q11. What is Div in HTML?**  
Ans: `<div>` is a block-level container used to group elements and apply styles.

---

**Q12. What is a View in SQL?**  
Ans: A view is a virtual table created from a SQL query. It does not store data itself but shows data from one or more tables.

---

**Q13. What does A stand for in ACID?**  
Ans: A stands for Atomicity. It means a transaction should either complete fully or not happen at all.

---

**Q14. What is a Void Element in HTML?**  
Ans: A void element does not have a closing tag. Examples: `<br>`, `<img>`, `<hr>`.

---

**Q15. Difference between Class and ID in CSS?**  
Ans: ID is unique and used for one element. Class can be used for multiple elements.

---

**Q16. What are Selectors in CSS?**  
Ans: Selectors are used to select HTML elements to apply styles. Examples: element selector, class selector, id selector.
