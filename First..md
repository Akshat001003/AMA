# AMA – 23/02/2026

**Q1. What is the difference between List and Tuple?**  
Ans: List is mutable and defined using []. Tuple is immutable and defined using ().

**Q2. What is a Class?**  
Ans: A class is a blueprint for creating objects. It contains attributes (variables) and methods (functions).

**Q3. What is Inheritance?**  
Ans: Inheritance allows a child class to use the properties and methods of a parent class. It helps in code reuse.

**Q4. What are SQL Joins?**  
Ans: Joins are used to combine data from multiple tables. Types include INNER JOIN, LEFT JOIN, RIGHT JOIN, and FULL JOIN.

**Q5. Why do we need DBMS?**  
Ans: DBMS is used for efficient data storage, reducing redundancy, providing security, and allowing multiple users to access data.

**Q6. What is the use of git add?**  
Ans: git add moves changes to the staging area.  
Example:
git add file.txt  
git add .

**Q7. How to create a new branch in Git?**  
Ans:
git checkout -b branch_name

**Q8. What are Libraries?**  
Ans: Libraries are collections of pre-written code that help perform specific tasks.

**Q9. What is Hashing?**  
Ans: Hashing converts data into a fixed-size value for fast lookup. It is used in dictionaries and for secure data storage.

**Q10. What is an Interpreted Language?**  
Ans: An interpreted language executes code line by line without a separate compilation step. Example: Python.

**Q11. What is the grep command?**  
Ans: grep is used in Linux to search for specific text inside files.  
Example:
grep "text" file.txt

**Q12. What is a Dictionary?**  
Ans: A dictionary stores data in key-value pairs. Keys must be unique.  
Example:
d = {"key": "value"}

**Q13. What is the difference between Class and Instance Variables?**  
Ans: Class variables are shared by all objects, while instance variables are unique for each object.

**Q14. What are break and continue statements?**  
Ans: break exits the loop completely, while continue skips the current iteration and moves to the next iteration.

**Q15. What is else with a loop?**  
Ans: The else block executes when the loop finishes normally without encountering a break.

**Q16. What is Multiple Inheritance?**  
Ans: Multiple inheritance allows a class to inherit properties and methods from more than one parent class.
